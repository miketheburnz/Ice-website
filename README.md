# Glacier Pure Ice

Vite + React starter for Glacier Pure Ice. This project is ready to deploy to Vercel.

Getting started:

1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build: `npm run build`

Vercel deployment:

- Import the repository to Vercel and use the default settings.
- Build command: `npm run build`
- Output directory: `dist`
